# Share scene

Scenes can be shared to external users that are not logged in.

* An url is generated for the scene.
* There is an url token that can be used to access the scene.
* The share link is valid for a configurable amount of time.

## Display page

* The title is formatted in this way: "Chapter I - Title: Scene title"
* The scene is rendered in formatted HTML.
* The description is inside a collapsed card
