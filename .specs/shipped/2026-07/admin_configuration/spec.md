# Admin Configuration

The admin configuration should have a sidebar listing the various configuration options.

* Settings: keep search engine settings, add a main title "general settings"
* Appearance and accessibility, which will later include graphical options, but can be created empty for now.
* Export and import, which will have tabs:
  * Export
  * Import
* Database configuration: will allow to switch between different databases, and to convert sqlite databases to mysql and vice versa.
